# BlinkIT-Grocery-Analysis
## Overview

Performed sales analysis on BlinkIT grocery data using Python to identify outlet performance, sales trends, and customer purchasing patterns.

## Tools Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn

## Key Insights

* Analyzed 8,000+ sales records
* Tier 3 medium outlets generated highest sales
* Identified sales trends by outlet size and item category

## Features

* Sales Analysis
* Outlet Performance Tracking
* Category-wise Analysis
* Data Visualization
